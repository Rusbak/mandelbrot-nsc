# L01 - Naive Python
L01 introduces the project structure and workflow. It covers setting up a Python environment, installing scientific packages, and learning Git for version control. At first the naive Mandelbrot is implemented to establish a baseline for later optimization. The topic highlights reproducibility, clear project structure, and progressive improvement through the project.
